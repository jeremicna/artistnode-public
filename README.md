# artistnode

`artistnode` is a small Node.js app for exploring artist relationship data and powering the Artistle game/API.

This public repo is mainly an archive of the app code and front-end experience.

The crawler and data-generation pipeline are intentionally not included here. I chose not to make the crawler public in order to reduce the risk of abuse.

As a result, this repository should be viewed as a public archive of the serving layer rather than the full system.
